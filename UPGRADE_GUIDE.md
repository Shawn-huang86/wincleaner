# WinCleaner 升级指南 v1.0.0 → v1.1.0

## 🚀 快速升级

### 方法一：使用升级脚本（推荐）
```bash
# 双击运行
build-upgrade.bat
```

### 方法二：手动命令
```bash
# 1. 升级版本
npm version patch

# 2. 清理旧构建
rm -rf dist dist-electron

# 3. 重新构建
npm run build && npm run dist-win
```
